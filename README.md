# My Business Khatabook 📒

A full-stack, responsive web application designed to help small businesses manage their daily transactions (Udhaar and Jama) digitally. Built with a focus on a clean user interface and real-time database synchronization.

## ✨ Features
* **Dashboard Overview:** Real-time calculation of Net Balance, Total 'Gave', and Total 'Got'.
* **Customer Management:** Add new customers and securely store their details.
* **Transaction Tracking:** Record daily credit and debit entries for each customer.
* **Dynamic UI:** Smooth view transitions, interactive modal popups, and automated toast notifications.
* **Complete CRUD Operations:** Create, Read, and Delete functionality for both customers and individual transactions, fully synced with the database.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript (Fetch API, DOM Manipulation)
* **Backend:** PHP (RESTful API approach)
* **Database:** MySQL
* **Icons & Styling:** FontAwesome, Custom CSS Animations

## 🚀 How to Run This Project
1. Clone this repository to your local machine.
2. Ensure you have a local server environment running (like WAMP, XAMPP, or MAMP).
3. Place the project folder inside your server's root directory (e.g., `www` or `htdocs`).
4. Open phpMyAdmin and create a new database named `khatabook`.
5. Import the provided `khatabook.sql` file into your new database to set up the tables.
6. Open your browser and navigate to `http://localhost/your-folder-name/khatabook.html`.
